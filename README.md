## Dashboard Overview

This dashboard provides end-to-end visibility into task execution and publishing status across both **FDL** and **legacy systems**.

## Key Features

Tracks all tasks that have been:

- Successfully published to **FDL**
- Successfully published to **legacy systems**
- Not published to **either FDL or legacy**

Enables quick identification of **missing, failed, or pending publishes**.

## Performance & Runtime Analysis

- Calculates the **average runtime** of tasks based on historical execution data.
- Analyzes trends over the **last two years** to identify long-running tasks and performance degradation.
- Supports **proactive optimization** and **capacity planning**.

## Value & Impact

- Improves operational monitoring and transparency.
- Reduces manual tracking and investigation effort.
- Enables data-driven decision-making.
